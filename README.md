## Getting Started

#### Install dependencies

```
yarn install
```

#### Compiles and hot-reloads for development

```
yarn dev
```

#### Compiles and minifies for production

```
yarn build
```

#### Run your unit/integration tests

```
yarn test:unit
```

#### Run your e2e tests

```
yarn test:e2e
```

#### Lints and fixes files

```
yarn lint
```
